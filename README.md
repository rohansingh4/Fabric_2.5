# Fabric 2.5

## Overview
Fabric 2.5 is a private Hyperledger Fabric network setup with:
- 2 channels
- 2 peers
- 2 orgs
- Private data using both channels and private data collections

## Project Goals
- Provide a minimal multi-org Fabric 2.5 network
- Demonstrate channel-based privacy and private data collections

## Quick Facts
- Network type: private
- Orgs: 2
- Peers: 2
- Channels: 2
- Privacy: channels + private data collections
